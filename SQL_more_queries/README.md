# SQL - More queries

This project focuses on advanced MySQL queries, user management, and database privileges.

## Learning Objectives

- How to create a new MySQL user
- How to manage privileges for a user to a database or table
- What's a PRIMARY KEY and FOREIGN KEY
- How to use NOT NULL and UNIQUE constraints
- How to retrieve data from multiple tables in one request
- What are subqueries
- What are JOIN and UNION

## Requirements

- All files executed on Ubuntu 20.04 LTS using MySQL 8.0
- All SQL keywords in uppercase (SELECT, WHERE, etc.)
- All files end with a new line
- All SQL queries have comments above them
- All files start with a comment describing the task

## Tasks

0. **My privileges!** - Lists all privileges of MySQL users user_0d_1 and user_0d_2
1. **Root user** - Creates MySQL server user user_0d_1 with all privileges
