# Python - More Classes and Objects

This project extends OOP concepts in Python through a series of tasks implementing a `Rectangle` class.

## Tasks

| File | Description |
|------|-------------|
| `0-rectangle.py` | Empty `Rectangle` class |
| `1-rectangle.py` | Private `width` and `height` with property getters/setters |
| `2-rectangle.py` | Adds `area()` and `perimeter()` methods |
| `3-rectangle.py` | Adds `__str__` for `#` representation |
| `4-rectangle.py` | Adds `__repr__` for `eval()` recreation |
| `5-rectangle.py` | Adds `__del__` with deletion message |
| `6-rectangle.py` | Adds `number_of_instances` class attribute |
| `7-rectangle.py` | Adds `print_symbol` class attribute |
| `8-rectangle.py` | Adds `bigger_or_equal` static method |
| `9-rectangle.py` | Adds `square` class method |
