# Python - Input/Output

This project covers file I/O and JSON serialization/deserialization in Python.

## Tasks

| File | Description |
|------|-------------|
| `0-read_file.py` | Read and print a UTF-8 text file |
| `1-write_file.py` | Write a string to a file, return char count |
| `2-append_write.py` | Append a string to a file, return char count |
| `3-to_json_string.py` | Serialize an object to a JSON string |
| `4-from_json_string.py` | Deserialize a JSON string to a Python object |
| `5-save_to_json_file.py` | Save a Python object to a JSON file |
| `6-load_from_json_file.py` | Load a Python object from a JSON file |
| `7-add_item.py` | Add CLI arguments to a JSON-backed list file |
| `8-class_to_json.py` | Return `__dict__` of an object for JSON serialization |
| `9-student.py` | Student class with `to_json()` method |
| `10-student.py` | Student class with filtered `to_json(attrs)` |
| `11-student.py` | Student class with `reload_from_json()` |
| `12-pascal_triangle.py` | Generate Pascal's triangle as a list of lists |
