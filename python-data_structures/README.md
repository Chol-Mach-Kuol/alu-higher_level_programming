# Python - Data Structures: Lists, Tuples

This project covers Python data structures including lists and tuples.

## Tasks

- **0-print_list_integer.py** - Print all integers of a list
- **1-element_at.py** - Retrieve an element from a list
- **2-replace_in_list.py** - Replace an element in a list
- **3-print_reversed_list_integer.py** - Print integers of a list in reverse
- **4-new_in_list.py** - Replace element in a copy of a list
- **5-no_c.py** - Remove all `c` and `C` characters from a string
- **6-print_matrix_integer.py** - Print a matrix of integers
- **7-add_tuple.py** - Add 2 tuples
- **8-multiple_returns.py** - Return length and first character of a string
- **9-max_integer.py** - Find the biggest integer in a list
- **10-divisible_by_2.py** - Find all multiples of 2 in a list
- **11-delete_at.py** - Delete item at a specific position in a list
- **12-switch.py** - Switch the values of two variables
