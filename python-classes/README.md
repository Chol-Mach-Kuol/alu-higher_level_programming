# Python - Classes and Objects

This project introduces Object-Oriented Programming (OOP) in Python through a series of tasks implementing a `Square` class.

## Tasks

| File | Description |
|------|-------------|
| `0-square.py` | Empty `Square` class |
| `1-square.py` | `Square` with private `size` attribute |
| `2-square.py` | `Square` with size validation (type and value) |
| `3-square.py` | `Square` with `area()` method |
| `4-square.py` | `Square` with property getter/setter for `size` |
| `5-square.py` | `Square` with `my_print()` method using `#` |
| `6-square.py` | `Square` with `position` attribute and offset printing |
