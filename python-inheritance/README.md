# Python - Inheritance

This project explores inheritance in Python through a series of tasks building up from simple lookup functions to a class hierarchy of geometry shapes.

## Tasks

| File | Description |
|------|-------------|
| `0-lookup.py` | Returns list of attributes/methods of an object |
| `1-my_list.py` | `MyList` subclass of `list` with `print_sorted()` |
| `2-is_same_class.py` | Checks exact class match using `type()` |
| `3-is_kind_of_class.py` | Checks instance using `isinstance()` |
| `4-inherits_from.py` | Checks strict subclass inheritance |
| `5-base_geometry.py` | Empty `BaseGeometry` class |
| `6-base_geometry.py` | `BaseGeometry` with unimplemented `area()` |
| `7-base_geometry.py` | Adds `integer_validator()` method |
| `8-rectangle.py` | `Rectangle` inheriting from `BaseGeometry` |
| `9-rectangle.py` | `Rectangle` with `area()` and `__str__` |
| `10-square.py` | `Square` inheriting from `Rectangle` |
| `11-square.py` | `Square` with its own `__str__` |
