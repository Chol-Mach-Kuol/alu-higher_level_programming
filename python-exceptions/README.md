# Python - Exceptions

This project covers Python exceptions, error handling, and the try/except/finally pattern.

## Tasks

- **0-safe_print_list.py** - Print x elements of a list using try/except
- **1-safe_print_integer.py** - Print an integer safely, return True/False
- **2-safe_print_list_integers.py** - Print only integers from a list
- **3-safe_print_division.py** - Divide two integers with finally debug
- **4-list_division.py** - Divide two lists element by element
- **5-raise_exception.py** - Raise a TypeError
- **6-raise_exception_msg.py** - Raise a NameError with a message
